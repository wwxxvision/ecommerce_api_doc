getProducts

URL

https://virtserver.swaggerhub.com/wwxxvision/Ecommerce/1.0.0/products

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    Лимит элементов на текущем запросе

    Показать элементы на текущей странице

    Параметр по которому будем фильтровать продукты [Brand, Rating, Price, Stock_Status, Tag, Attribute]

    Параметр по которому будем сортировать продукты [Price, Date, Rating]
